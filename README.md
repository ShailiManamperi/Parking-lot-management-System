## 🔧 Installation

### Clone Repository

```bash
git clone https://github.com/ShailiManamperi/Deparment-Management-Website.git
cd Department-management-website
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
DB_HOST=your-host
DB_PORT=your-port
DB_USER=your-user
DB_PASSWORD=your-password
DB_NAME=your-database
```

### Run Project

```bash
npm start
```
